# 💗 Projeto Romântico

Um site romântico desenvolvido com HTML, CSS e JavaScript, criado com o objetivo de proporcionar uma experiência emocional, leve e visualmente agradável.

O projeto foi pensado para ser simples, intuitivo e ao mesmo tempo marcante, com foco em design moderno e interação suave.

---

## ✨ Funcionalidades

* 💗 Página inicial com palavras carinhosas aparecendo dinamicamente
* 📖 Seção "Nossa história" com narrativa emocional em blocos
* 🎁 Página surpresa com:

  * Alerta interativo
  * Texto com efeito de digitação
  * Card com efeito visual moderno (glassmorphism + leve 3D)
* 🎨 Design com cores suaves (rosa e branco)
* 🧠 Navegação simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (Flexbox, animações, efeitos visuais)
* JavaScript (DOM, interações, setTimeout)

## 🚀 Como executar

1. Baixe ou clone o repositório:
   git clone https://github.com/carlos-gabriel-124/Perfil-Romantico.git

2. Abra o arquivo:
   index.html

3. Pronto! O site já estará funcionando no navegador.
